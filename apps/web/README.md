# @sukoon/web

The Sukoon web app: the lowest-friction way to try Sukoon (upload → clean → download) and the SEO
landing page that drives installs of the heavier platform apps.

Processing happens **in the browser, on-device** — MDX-Net runs client-side via WASM, so the file
never leaves the device.

## Status

Planned (scaffold present).

## Develop

```bash
pnpm install
pnpm --filter @sukoon/web dev
```
